## 7) Passo a passo completo

1. **Duplicar a página atual.** Instale o plugin **"Yoast Duplicate Post"** (ou "Duplicate Page") → na lista de Páginas, passe o mouse na sua página de vendas → **"Clonar"**. (Ou no Elementor: abrir a página → ☰ → "Salvar como modelo" → inserir numa página nova.)
2. Renomeie a página para **"Nutricionista Esportivo"**.
3. **Edite com Elementor** e troque os textos conforme as seções 1 a 5 acima.
4. Defina o **slug** `nutricionista-esportivo` (seção 6).
5. Preencha **Título SEO + Meta description** no plugin de SEO (seção 6).
6. **Publicar.**
7. **Search Console:** abra a propriedade → "Inspeção de URL" → cole a URL nova → **"Solicitar indexação"** (acelera o Google achar a página).
8. **Link interno:** coloque um link pra essa página no menu do site ou na home (ajuda o Google e os visitantes a encontrarem).
9. Pronto — agora é aguardar o Google indexar e ranquear (semanas).
